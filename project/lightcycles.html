Lightcycles is a retro-inspired game developed for a school project, drawing inspiration from the Tron franchise. Built using Raylib for rendering and C++ as the core language, the objective was to create a fun, portable multiplayer experience.

Up to 4 players can compete against one another on a single computer or challenge up to 7 AI opponents, with fully customizable speed and turn settings, Lightcyles captures the fast-paced and strategic gameplay of classic light cycle battles.
