Coldblood [Unity, PUN2, C#]

Coldblood is a first-person, realistic multiplayer combat simulator and solo passion project focused on immersive character control and advanced weapon customization.

Developer

Nathan Hillger

Publisher

None

Repo

The game features a fully dynamic character controller integrated with the Unity Animation Rigging (IK) library, enabling precise animation control and real-time constraints.

With this system, players experience realistic body movement, including dynamic leaning, crouching, and prone transitions with smooth positional adjustments rather than fixed heights. The weapon customization system allows attachments to be applied at any position, with animations dynamically adapting in real time.

Utilizing Photon Unity Networking 2 (PUN2), Coldblood is fully networked, supporting thousands of interactive items on the server simultaneously. Each item is interactable as if physically present in the world, creating a deeply immersive multiplayer experience.
